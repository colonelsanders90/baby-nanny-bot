Common questions

FAQ

Do I need to download an app?
No. Nanny Bot runs entirely inside Telegram, which you almost certainly already have. There is nothing to install beyond what you've already got.
Is my data private?
Yes. The bot only responds to chat IDs you explicitly authorise. Your feeds and nappy data are stored in a private database and never shared with any third party. The source code is open on GitHub so you can verify this yourself — or self-host on your own infrastructure.
How do I share tracking with my partner?
Send /share in your chat — the bot replies with a one-time link code. Your partner sends /join <code> in their own chat with the bot. From that point on, every log either of you makes is shared and visible to both of you instantly.
Is it designed for bottle-fed babies?
Yes — feed tracking is built around ml volumes. It works best for bottle feeding (formula or expressed breast milk) where you know the amount. If you're breastfeeding, you can still use nappy tracking and the timing charts; the ml features just won't be relevant.
What if I logged the wrong amount or time?
Tap /delete (or the 🗑️ Delete button from the menu), choose the entry from the list, and confirm. The entry is removed. Then log the correct one. Times up to 12 hours in the past can be set when logging, so you can backdate if you forgot to log at the time.
Can I run this myself?
Absolutely — it's open source under MIT. You'll need a Telegram bot token from @BotFather, a PostgreSQL database, and a server (Railway is the easiest option). Full setup instructions are in the GitHub repo.
